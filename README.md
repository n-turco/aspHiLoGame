# ASP Hi-Lo Game

## Project Overview

ASP Hi-Lo Game is a web-based gaming application developed with ASP.NET that challenges players to guess a randomly generated number. This project emphasizes the importance of input validation and secure data handling in web applications.

## Key Features

- Random number generation with guessing game mechanics
- Strict input validation to prevent invalid entries
- Real-time game state management
- Responsive web interface
- Secure data handling throughout the application

## Technical Stack

- C#
- ASP.NET Core
- HTML5
- CSS3

## Security Enhancements

- ASP.NET validation to prevent rogue data entry
- Strict adherence to expected input formats
- Secure session management across requests

## How to Run (Local Development)

1. Ensure .NET runtime is installed on your system
2. From the project folder, restore and build:

   ```
   dotnet restore
   dotnet build
   ```

3. Run the application:

   ```
   dotnet run
   ```

4. Open a browser and navigate to `https://localhost:5001` (or the port printed by the app)
