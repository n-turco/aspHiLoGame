aspHiLoGame
Overview:
A web-based game developed with ASP.NET that challenges players to guess a randomly generated number. This project emphasizes the importance of input validation and secure data handling in web applications

Technical Features:
Security & Validation: Utilized ASP validation to prevent rogue data entry and ensure strict adherence to expected input formats

Game Logic Architecture: Engineered the backend logic to handle state across multiple web requests, reinforcing a deep understanding of ASP.NET

Responsive Design: Optimized the front-end using HTML/CSS for a seamless user journey

Technologies: C#, ASP.NET, HTML, CSS
